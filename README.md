# file-server
A simple command-line FTS (file transfer system) for linux


Written in python


Makefile not required

# Commands:
  
  ls [path]
  
  cd  [path]
  
  pwd
  
  mkdir  [path]
  
  rm  [-r]  [path]
  
  get  [-r]  [path]
  
  
  put  [-r]  [local-path]
  
  lls  [local-path]
  
  lcd  [local-path]
  
  lpwd
  
  lmkdir [local-path]
